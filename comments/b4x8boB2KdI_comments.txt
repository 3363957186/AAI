    â€‹@kpsayyed84  "Binning" refers to metaphorical (or maybe literal) bins, as in containers, used to sort the chips by performance capability.

All processor dies get tested for the top spec, those that pass go in the top bin.
Those that fail get tested for the next spec down, those that pass go into that bin.
